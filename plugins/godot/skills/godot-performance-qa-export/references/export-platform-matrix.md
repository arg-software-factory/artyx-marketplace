# Export platform matrix

Create a named export preset for every supported platform, install matching export templates, and
test the startup scene in an exported build. Audit resource filters, feature tags, icons, signing,
permissions, renderer compatibility, input, paths, and memory budget per target. Never assume an
editor-only asset or absolute file path exists in a packaged game.

- Exporting projects: https://docs.godotengine.org/en/4.6/tutorials/export/exporting_projects.html
