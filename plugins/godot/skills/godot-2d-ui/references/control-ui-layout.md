# Control layout and themes

Use Control anchors for proportional placement and Containers for layout; do not fight a container
with manual child positions. Give size flags, minimum sizes, and stretch ratios semantic meaning.
Build reusable visuals with Theme resources and theme type variations instead of setting every
widget's colors or fonts individually.

Keep focus order and keyboard/controller navigation usable. Let Controls receive input before
gameplay; use accessible contrast, readable scaling, and localized text expansion in layout tests.
