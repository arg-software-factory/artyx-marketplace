# Figma token

Create or confirm a Figma token and provide it as `FIGMA_TOKEN` when prompted.

## Steps

1. Verify Figma's current MCP authentication requirements in the official docs.
2. Create or locate the token required for remote MCP access.
3. Paste the token into Artyx when prompted for `FIGMA_TOKEN`.

Docs: https://help.figma.com/hc/en-us/articles/32132100833559-Guide-to-the-Dev-Mode-MCP-Server
