# Figma

Authorize Figma and enable its MCP server. No local in-app Artyx add-on is
needed.

## Steps

1. In Figma, enable the MCP / Dev Mode MCP server if required by your plan.
2. Click Sign in and authorize when prompted.

Docs: https://help.figma.com/hc/en-us/articles/32132100833559-Guide-to-the-Dev-Mode-MCP-Server
